# Бесконечная итерация

В этом задании практикуется шаблон *Iterator* (*Итератор*). Написан класс, расширяющий `Iterable<Integer>`, по которому мы будем итерироваться. В процессе итерирования мы будем от него получать случайные числа в диапазоне значений. Диапазон задаётся двумя числами - минимальным и максимальным значениями. Передаются они через параметры конструктора. Таким образом, итератор по объектам нашего класса будет итерироваться по бесконечной последовательности из случайных чисел в заданом интервале.

Пример использования нашего класса:
```java
public static void main(String[] args) {
  for (int r : new Randoms(90, 100)) {
    System.out.println("Случайное число: " + r);
    if (r == 100) {
      System.out.println("Выпало число 100, давайте на этом закончим");
      break;
    }
  }
}
```
